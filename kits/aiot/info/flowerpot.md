# 智能澆花裝置

大千世界，興趣各異，愛花花草草之人不乏。但難得出一趟遠門就不得不難為它們，時間長一點更是無法輓救。所以智能澆花裝置是必不可少，另外還帶有遠程報信功能，澆花完成通過郵件方式告知，另你更安心。

### 程式流程

1. 將Micro:bit程式上載到Micro:bit。
2. 等待Wifibrick連上網絡。
3. 當泥土土壤濕度太低，灑水器會自動啟動，並通知伺服器。
4. IoT程式的小貓會說出灑水次數。
