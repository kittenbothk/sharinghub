# 智能氣象站套件

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

智能氣象站是以天氣監測為主題的學習套件，一套套件可以做齊亮度檢測、雨量檢測、溫濕度檢測和風向風速檢測的功能。套件包含的積木可以搭建出一個包含所有功能的完整氣象站模型或者單獨功能五個小模型。
