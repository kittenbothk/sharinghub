# KOI 2 自動導航泊車套件

<figure><img src="../../.gitbook/assets/鄧顯 小車.png" alt=""><figcaption></figcaption></figure>

配合KOI 2的路牌追蹤功能，Kittenbot設計了可以按照路牌自動完成倒後泊車操作的機械小車套件。

## 套件特色

* 仿真機械車底盤
  * 配有差速器傳動系統及仿真轉向結構
* 自動導航駕駛
  * 透過辨認路牌可以做到自動泊車的操作

## 套件內容

* Robotbit 2.2擴展板 \*1
* KOI 2 AI鏡頭 \*1
* Geekservo 2KG 馬達 \*1
* Geekservo 9G 舵機 \*1
* 積木包 \*1
* 18650鋰電池 \*1
