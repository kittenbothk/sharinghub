# 人面追蹤風扇

![](../../../.gitbook/assets/温控风扇.png)

使用人面追蹤功能追蹤人面，自動將風扇指向人面。另外可以透過語音辨識調較風扇的速度。

{% hint style="info" %}

{% endhint %}

1. 連接好Micro:bit和打開Robotbit電源
2. 點擊綠色旗啟動程式
3. 系統會自動追蹤鏡頭裡面的人面，並將風扇指向人面。
