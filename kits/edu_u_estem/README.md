# 教大E-Stem Kit

為教大課程與教學部門設計的E-Stem學習套件

## 套件內容

* 未來板Lite AI
* 超聲波感應器 x2
* 磁力感應器
* 環境氣壓感應器
* PM2.5懸浮粒子感應器
* 水溫感應器
* 聲音感應器
* Geekservo紅色電機
* 導電性檢測模組
* 連接線
* 風速儀積木包
* 速度計積木包
* 火箭發射積木包
