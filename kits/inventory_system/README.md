# 工業4.0倉庫貨物管理系統

<figure><img src="https://kittenbothk.readthedocs.io/en/latest/_images/belt.png" alt=""><figcaption></figcaption></figure>

工業4.0倉庫貨物管理系統模擬了倉庫內貨物進出的情景，透過感應貨物的RFID晶片，系統可以將進出的貨物資料上傳到物聯網平台作管理。
