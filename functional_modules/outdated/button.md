# 數碼按鈕

![](https://kittenbothk.readthedocs.io/en/latest/\_images/button\_1.png)

這是一隻按鍵模組，輕易為你的Microbit提供額外按鍵。

### 產品參數

* 工作電壓：3V\~5V
* 接口：杜邦線
* 反饋數字：未按下返回1，按下返回0
