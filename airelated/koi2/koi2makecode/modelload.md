#### 程式解說

{% hint style="info" %}
成功載入模型時，KOI會顯示Load Successfully的畫面。

載入到模型後KOI就會自動進行圖像辨識。
{% endhint %}

1. 按A鍵載入模型。
2. 按B鍵進行辨識，Micro:bit會顯示辨識結果以及相似值(0-100)。

[參考程式](https://makecode.microbit.org/_0VP8RzDaiLXu)

#### 程式解說

{% hint style="info" %}
成功載入模型時，KOI會顯示Load Successfully的畫面。

載入到模型後KOI就會自動進行圖像辨識。
{% endhint %}

1. 按A鍵載入模型。
2. 按B鍵進行辨識，Micro:bit會顯示辨識結果以及相似值(0-100)。
3. 假如辨認結果為A，Micro:bit會顯示心形圖案；假如辨認結果為B，Micro:bit會顯示笑臉圖案。
