### 參考程式

{% hint style="info" %}
錄音與播放功能需要高速Micro SD卡(class 10或以上)，KOI原廠不配備SD卡，有需要的請自行預備。
{% endhint %}

{% hint style="info" %}
Armourbit用家請使用初始化Armourbit積木。
{% endhint %}

#### 程式解說

1. 按下B鍵，KOI會進行3秒的錄音並儲存為testing.wav。
2. 按下A鍵，KOI會播放SD卡中名為testing.wav的檔案。
